# Contribuir al Proyecto SubCreator

¡Gracias por considerar contribuir a **SubCreator**! Estamos encantados de recibir aportes de la comunidad para mejorar este proyecto.

---

## Cómo contribuir

### 1. Reportar un problema
Si encuentras un error o tienes una idea para una nueva funcionalidad, por favor abre un **issue** en el repositorio oficial:  
[https://github.com/wpadillav/SubCreator/issues](https://github.com/wpadillav/SubCreator/issues)

Cuando crees un **issue**, proporciona tanta información como sea posible:
- Descripción clara del problema o mejora.
- Pasos para reproducir el problema (en caso de ser un bug).
- Logs relevantes, si los hay.
- Detalles de tu entorno (SO, versión de Apache, etc.).

### 2. Mejorar la documentación
La documentación es una parte importante del proyecto. Si encuentras algún error o algo que podría explicarse mejor, no dudes en enviar un **pull request** con las mejoras.

### 3. Contribuir con código
Si quieres contribuir con código:
1. **Haz un fork del repositorio.**
